## Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-20

### Added
- Real-time Discord status integration with Lanyard API
- Live Spotify activity display with album artwork
- Game activity tracking and display
- Animated gradient orbs background effect
- Glass morphism card design with backdrop blur
- Dark theme optimization with CSS variables
- Comprehensive documentation updates
- Security policy guidelines

### Changed
- Migrated from static HTML to Next.js 14 framework
- Enhanced responsive design for all devices
- Improved color scheme for better accessibility
- Updated all dependencies to latest versions
- Refined typography with Inter and JetBrains Mono fonts
- Optimized image loading with Next.js Image component

### Fixed
- Cross-browser compatibility issues
- Mobile responsiveness on small screens
- Font loading performance
- Icon alignment in footer links
